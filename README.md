# scintax

Scintax is a toy language based on Lox that follows JavaScript syntax.

# Namespaces

* scintax - scintax core parser and lexer
* simp - scintax interpreter & mathematical processor
* ssh - scintax semantic highlighter
* scat - scintax cat
* sed - scintax editor
